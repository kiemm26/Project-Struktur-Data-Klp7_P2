# Duplicate Data Detection System

This project implements a simple duplicate detection system using C++.

Features:
- CSV dataset processing
- Hash table based duplicate detection
- Performance benchmarking
- File output for analysis

Dataset:
Online Retail II Dataset (UCI Machine Learning Repository)
https://archive.ics.uci.edu/dataset/502/online+retail+ii

Structures used:
- unordered_map (hash table)
- vector

Course:
Data Structures Project
